# Hives & Colonies

Versión: 0.1.0-alpha

## Descripción

Hives & Colonies es un modpack centrado en la construcción de una civilización mágica mediante MineColonies, Ars Nouveau y Productive Bees.

## Estado

Esta es una versión Alpha temprana.

Características implementadas:

* MineColonies
* Ars Nouveau
* Productive Bees
* Create
* Waystones
* Primeras recetas balanceadas
* Primeras quests

## Feedback

Si encuentras errores, problemas de balance o sugerencias, repórtalos para futuras versiones.

## Requisitos

Minecraft 1.21.1
NeoForge
8 GB de RAM recomendados
